    Welcome to Dr.Z Network

    Dr.Z Network is a CQA Platform which aims to find an effectively ranked list of answers as well as some cold-start experts to provide new answers to the questions. When a new question enters our system, the Question Analysis function prepares the question for three tasks, which are the Question Similarity, the Tag Propagation and the Cold-Start Expert Finding. The Cold-Start Expert Finding receives the question and effectively finds cold-start experts to answer the question. The Tag Propagation analyzes the question and generates a list of tags for the question. The Question Similarity utilizes the raw question as well as the tags generated by Tag Propagation to search for similar questions in the database finding candidate answers. The answers along with the raw question are passed to the Answer Ranking function which includes two parts, the Feature Extraction and the Ranking. It will output an effectively ranked list of final answers.

    Our Services

    Web Design

    The incompleteness of tags of questions limits more application on the system. It is important that recommend multiple tags to a question. We apply the knowledge about similarity and SNS to build the network among questions and tags effectively.

    Experts Finding

    When a new user first arrives to a CQA system, no previous activity is available and he/she is said to be at a cold-start. In our system we want to find some skillful cold-start experts that can answer the questions.

    Answer Ranking

    In this project, we aimed to achieve efficient Answer Ranking by utilizing NLP techniques such as Part-Of-Speech tagging, parsing and Word Net relatedness as well as ML classification techniques, SVM, Naive Bayes and KNN to learn to rank the answers.
